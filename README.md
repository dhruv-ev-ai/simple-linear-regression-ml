# 📈 Simple Linear Regression Salary Predictor

This project demonstrates a **Simple Linear Regression model** that predicts salary based on years of experience.  
The project also includes an **interactive Gradio interface** that allows users to enter experience and get a predicted salary.

---

## 📌 Project Overview

The goal of this project is to build a machine learning model that learns the relationship between:

- Years of Experience
- Salary

The model is trained using **Simple Linear Regression** and deployed with a **Gradio interface** for easy prediction.

---

## 📊 Dataset

Dataset used:

Salary vs Experience dataset

Features:

- Experience (Years)
- Salary

This dataset is commonly used to demonstrate regression algorithms.

## 🧠 Machine Learning Model
